# Blinkit Clone - Flutter Grocery App

A premium, full-feature Blinkit-style grocery application built with Flutter. This project features a modern UI, product management, and a robust offline-first database system.

## 🚀 Repository Link
[https://github.com/pratiknagap5-cpu/blinkitclone](https://github.com/pratiknagap5-cpu/blinkitclone)

## 🌐 Live Demo (Web)
[https://pratiknagap5-cpu.github.io/blinkitclone/](https://pratiknagap5-cpu.github.io/blinkitclone/)

## ✨ Features
- **Modern UI**: Polished, responsive design inspired by the Blinkit app.
- **Product Categories**: Browse products by categories with a clean layout.
- **Cart Management**: Real-time cart updates using Provider.
- **Location Services**: Integrated location tracking for delivery addresses.
- **Database**: efficient data handling with SQFlite.
- **Image Fallback System**: Robust handling for network images with placeholders.

## 🛠️ Tech Stack
- **Framework**: Flutter
- **State Management**: Provider
- **Storage**: SQFlite (Local Database), Shared Preferences
- **Icons**: Cupertino Icons, Material Icons

## 📦 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/pratiknagap5-cpu/blinkitclone.git
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run
   ```
